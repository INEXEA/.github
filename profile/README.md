# INEXEA's GitHub Organization

Basic GitHub and related information on [INEXEA](https://www.inexea.co.uk/)

## Becoming a Member of this Organization

INEXEA employees who would like to join this GitHub organization must request an existing organization member - a colleague who is already a member - to submit an access request on their behalf.

Please ensure you meet the membership criteria below before requesting access.

## Membership Criteria

To request membership of this GitHub organization, please provide:

- Your INEXEA GitHub username (with multi-factor authentication (MFA) enabled).
- Your full name, as listed in the employee directory.
- The full name of your practice, unit or team.

## Work at INEXEA

Career opportunities at [INEXEA](https://inexea.co.uk/) are typically made available at the [careers section](https://uk.inexea.com/careers/) of our website.
